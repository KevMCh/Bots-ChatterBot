# _OperBot_

## Example DialogFlow

### Description

Example of chatbot using [DialogFlow](https://dialogflow.com/). [DialogFlow](https://dialogflow.com/) is the google plataform to create bots.

The functionality of this bot is do simple mathematics operations.

  * Add

  * Subtract

  * Multiply

  * Divide
